# CYBER-MD

**A WhatsApp Bot powered by Baileys**  
Created by **Keith-Tech**

## Features
- Owner-only access (Owner: `263789745277`)
- Basic commands:
  - `menu`
  - `ping`
  - `alive`
  - `repo`
- Uses `@whiskeysockets/baileys`
- Easy deployment on Render

---

## How to Deploy on Render

1. **Upload the bot files to a GitHub repo**
2. Go to [https://render.com](https://render.com) and log in or sign up
3. Click **"New Web Service"**
4. Connect your GitHub and select the repo
5. Set the following in the deploy settings:
   - **Environment**: Node
   - **Build Command**: *(leave blank or use)* `npm install`
   - **Start Command**: `node index.js`
6. Click **Deploy** — check the logs for a QR code to scan with WhatsApp
7. Once paired, you're online!
